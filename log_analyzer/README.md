### Заголовок
Nginx Log Analyzer — инструмент для анализа логов Nginx и генерации HTML-отчёта со статистикой по запросам.

### Описание
Цель проекта — автоматизировать разбор access-логов Nginx, определить наиболее нагруженные и медленные URL, а также представить результаты в удобном HTML-отчёте.

Основные функции проекта:
* поиск последнего лог-файла Nginx по имени;
* поддержка обычных файлов и сжатых .gz;
* парсинг строк логов (извлечение URL и времени обработки запроса);
* расчёт:
  * количества запросов;
  * суммарного времени;
  * среднего времени;
  * медианы;
  * максимального времени;
  * процента по количеству запросов;
  * процента по суммарному времени;
* генерация HTML-отчёта по шаблону templates/report.html.

### Установка
Требования:
* Python 3.9+
* Установленный менеджер пакетов uv

Установка зависимостей:
uv pip install -e .
uv pip install ruff mypy pytest pytest-cov structlog pre-commit

### Использование
1. Настройте конфигурационный файл config/config.json.
2. Запустите анализ логов:
   uv run python -m log_analyzer.log_analyzer --config config/config.json
